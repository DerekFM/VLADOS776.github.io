Creative & Survival Modes

Creative Mode

Creative Mode is one of the Game Modes in Scrap Mechanic. As of the current release, it is the only available mode.

Features

  • No hostile enemies.
  • Infinite parts, blocks, and machine parts, allowing for practicing building.
  • All basic building functionality available.

Survival Mode

Survival Mode is an upcoming Game Mode in Scrap Mechanic.

Backstory

The player character is a robot maintenance mechanic, sent to an automated agricultural planet to look after the worker robots. The mechanic's spaceship crash-lands on the planet, leaving them stranded. Quickly, they discover that the Agribots have gone haywire, attacking the mechanic on sight.

Known Features

The following features have been mentioned in devblogs, YouTube videos, or the TIGSource forum post.

  • Agribots
  • Consumable food
    • Corn
  • Start game in a crashed spaceship
    • Starting items obtained from the crashed spaceship
  • Obtain items by scavenging generated structures
  • Farming

Speculated or Implied Features

These features have not been officially mentioned, but are strongly suspected to be a part of Survival mode.

  • Player health
  • Limited inventory
  • Stackable items
  • Crates can be used to store items
  • Gas Engine and Electric Engine require fuel to function
